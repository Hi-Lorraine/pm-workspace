# 鼎新比价助手 — 商业化变现方案

> 核心认知：比价工具是获客漏斗，不是产品本身。免费工具抢入口，用户来了之后多层变现。

---

## 变现全景图

```
         免费工具（比价）
              ↓
         用户涌入1药城
              ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
  订单转化   数据沉淀   用户习惯养成
  （赚差价） （卖情报） （离不开你）
    ↓         ↓         ↓
    └─────────┼─────────┘
              ↓
      付费功能 + 流量变现
      （规模化盈利）
```

---

## 第一层：订单转化（最直接，现在就能做）

用户用比价工具查价 → 发现1药城价格差不多甚至更低 → 直接在1药城下单。

变现方式：

| 方式 | 说明 |
|------|------|
| 1药城自营利润 | 用户看完比价直接在1药城买，赚差价 |
| 价格策略调整 | 看到竞品价格后精准调价，该降的降，该守的守 |
| 凑单引导 | "再加 ¥80 就免运费"、"满 500 减 50"，提高客单价 |

产品设计已支持：
- 1药城便宜 → "已为您加入购物车" → 直接转化
- 竞品便宜 → 给链接但也给1药城购物车 → 仍有机会转化

## 第二层：数据变现（3-6个月，数据积累后）

### 独家数据资产

| 数据 | 价值 |
|------|------|
| 用户在搜什么药 | 需求洞察 |
| 各平台实时价格 | 竞品情报 |
| 用户最终在哪下单 | 转化分析 |
| 采购频次和规模 | 客户画像 |

### 卖给谁

| 买家 | 卖什么 | 怎么收费 |
|------|--------|---------|
| 药品厂家/经销商 | 市场情报："你的阿莫西林在各平台价格分布，全国多少药店在搜" | 情报订阅，按月收费 |
| 1药城采购部 | 竞品价格监控报表，辅助定价决策 | 内部价值，降本增效 |
| 药店老板 | "你的采购成本比同区域药店高 15%，主要贵在哪" | 高级分析报表，会员付费 |

## 第三层：流量变现（用户规模起来后）

掌握药店老板的"采购入口"= 掌握流量。

| 模式 | 做法 | 收入 |
|------|------|------|
| 竞价排名 | 供应商付费让价格排前面（标注"广告"） | CPC/CPM |
| 推荐位 | "今日特惠供应商" | 坑位费 |
| 新品推广 | 厂家出新药，精准推送给目标药店 | 按曝光/点击收费 |
| 独家优惠 | 和供应商谈专属折扣，只在工具里展示 | 佣金分成 |

⚠️ 注意：不能让广告影响比价客观性，"极致透明"是底线。

## 高粘性场景补充

除了比价，还有这些场景可以增加用户粘性：

### 第一梯队（和比价强关联）

| 场景 | 说明 | 粘性 |
|------|------|------|
| 智能补货提醒 | 库存快没了主动提醒，附带当前最低价 | 每天都要看 |
| 历史价格走势 | 过去30天价格曲线，判断"现在该不该买" | 每次下单前看 |
| 促销/满减提醒 | "还差 ¥120 凑满减，要不要加一箱板蓝根" | 帮用户省钱 |

### 第二梯队（采购决策痛点）

| 场景 | 说明 |
|------|------|
| 效期预警 + 近效期捡漏 | 提醒快过期的库存 + 推荐近效期低价药 |
| 采购清单一键比价 | 上传30-50种药的清单，自动最优分配方案 |
| 替代药推荐 | 断货时推荐同规格其他厂家的替代品 |

### 第三梯队（长期壁垒）

| 场景 | 说明 |
|------|------|
| 采购数据报表 | 月度采购分析，数据越积累越有价值 |
| 同行采购参考 | 同区域同规模药店的采购行为对比 |

### 优先级建议

| 优先级 | 场景 | 理由 |
|--------|------|------|
| ⭐ 现在 | 单品比价 | V1 核心，已在做 |
| ⭐ 紧跟 | 采购清单批量比价 | 真实工作流 |
| ⭐ 紧跟 | 促销/满减提醒 | 打开率高 |
| 🔜 下阶段 | 历史价格走势 | 需要数据积累 |
| 🔜 下阶段 | 智能补货提醒 | 需对接进销存 |
| 📋 远期 | 采购报表 | 数据壁垒护城河 |
